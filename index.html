Research Threads

Credential Resolvers (Agents and Accounts)

Resolver-based credential discovery for autonomous agents and controlled accounts. This thread focuses on how ENS resolvers can expose verifiable authorization data backed by ERC-8092 associations, without introducing new registries or trust assumptions. Two concrete instantiations are Agent Delegations and CCResolver.

  • Agent Delegations ERC - extends ERC-8092
  • CC Resolver demo for ERC-8092

Interoperable Addresses and Names

Cross-chain identity and discovery using ERC-7930 and ERC-7828 with ENS-backed chain registry resolution. This work is informed by hands-on development and personal contribution to the Ethereum Chain Registry, including core resolver logic and registry tooling.

  • Human readable chain specific addressing via ENS.
  • Official Ethereum Chain Registry

Execution and Mempool Tooling

A tool to visualize and compare transaction pools across different Ethereum execution clients. This helps debug transaction propagation issues, analyze mempool behaviors during attacks, and understand differences in transaction acceptance policies between clients.

  • txpool-viz

x402 Payments and Identity

Researching and shaping the agentic web payment layer (x402): request/settle messaging, identity-aware routing, and interoperable payTo schemes like ENS.

  • ENS payTo support for x402.
  • Identity metadata extension for messages.
