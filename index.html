Research Threads

Agent Delegations and Trustless Agents

Problem space: enforceable authorization for autonomous agents using delegable identity primitives and resolver-mediated discovery.

  • ERC-8092 based delegation.
  • ENS resolvers as credential surfaces.
  • CCIP Read for agent data.
  • AgentDelegations repo
  • Thesis section

Chain Registry and Interoperability

Problem space: cross-chain discovery and registry design using resolver-based composition and standards-driven identifiers.

  • ERC-7930 and ERC-7828.
  • ENS-based chain registry patterns.
  • Resolver-based discovery.
  • Notes and repos

Execution and Mempool Research

Problem space: transaction lifecycle observability and execution-layer visibility for protocol analysis.

x402 and Agent Payments

Problem space: standard payment request and settlement messaging for agents, with optional identity metadata and resolver-based routing.

  • ENS identity as a supported payTo scheme for x402.
  • Identity metadata extension for PaymentRequired and PaymentPayload.
  • Agent-to-agent payment flows aligned with decentralized identity and discovery.

  • ENS payTo integration PR
